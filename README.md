# 📌 Placement Application Tracker

A simple and responsive web application to **track your job or placement application statuses** (Applied, Interview, Selected, Rejected). It helps users organize and view all their job application progress in one place.  

👉 **Live Demo:** [Click here](https://eswargutala.github.io/placement-application-tracker/)

---

## 📝 Features
- Add and view placement or job applications
- Filter applications by status (Applied, Interview, Selected, Rejected)
- Clean and user-friendly interface
- Works on mobile and desktop browsers

---

## 🛠️ Technologies Used
- **HTML** – Structure of pages
- **CSS** – Styling and layout
- **JavaScript** – Logic for filtering and app interaction

---

## 🚀 How to Use
1. Open your browser.
2. Go to the **Live Demo** link above.
3. Add companies and roles to track your application statuses.
4. Use the filter to view applications by status.
